# COME359 Internet Programming Class Homeworks

## Course Objective

The purpose of this course is to provide knowledge about the advanced challenges of web programming. Students who successfully complete this course are expected to know the fundamentals of creating websites. Students will develop, manage, and maintain professional websites using HTML5, CSS, JavaScript, and ASP.NET MVC. The entire production process is covered, including pre-production planning, layout and design considerations (including site maps and navigation), site testing, deployment and implementation, maintenance, and current legal and ethical issues.

## Course Content

Introduction to Web Programming, HTML, CSS, JavaScript, JavaScript and DOM Events, ASP.NET MVC 5, Model-View-Controller, Web Forms and HTML Helpers, Data Annotation and Validation, Membership, Authorization and Security, AJAX and jQuery, Routing, Web API, Unit Testing.

### WEEK 1

**Topics Covered**: Internet fundamentals, web protocols (HTTP/HTTPS, TCP/UDP, DNS), browser functionality, frontend vs backend concepts, data formats (JSON/XML), RESTful APIs, security basics (HTTPS, cookies), HTML structure and forms, CSS selectors and styling, HTML5 features, responsive design, network protocols (FTP, SMTP, POP3, DHCP), error codes, and modern web technologies (WebSockets, CDN).
